# RecipeShare

A full-stack web application for sharing cooking recipes.

## MVP Demo Video
https://drive.google.com/file/d/15WQ96i9EaNRR9D60zkb7x61fb_5EB1n5/view?usp=drive_link

## Technologies Used
- Frontend: React
- Backend: Node.js, Express.js  
- Database: PostgreSQL
- Deployment: Render

## Features
- Create new recipes
- View all recipes
- Delete recipes
- Data persistence
